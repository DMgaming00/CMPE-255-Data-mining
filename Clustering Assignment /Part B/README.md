# Part B – Hierarchical Clustering

This folder contains all materials for hierarchical (agglomerative) clustering.

## Contents
- Dataset/
- Notebook/
- Report/
- Screenshots/

## Description
This part performs hierarchical clustering using Ward’s method.  
It includes:
- dendrogram visualization
- AgglomerativeClustering model
- cluster assignment
- silhouette score
- PCA visualization of clusters

This shows how clusters merge progressively in a hierarchy.
