# Part D – DBSCAN Clustering (sklearn Implementation)

Contains all resources for density-based clustering using DBSCAN.

## Contents
- Dataset/
- Notebook/ (Python 3.12–safe version using sklearn)
- Report/
- Screenshots/

## Description
This part replaces PyCaret with sklearn to avoid Python 3.12 issues.  
It performs:
- data scaling
- DBSCAN clustering
- noise detection
- silhouette score (when valid)
- PCA visualization

DBSCAN identifies clusters of arbitrary shape and detects outliers naturally.
